What is a Neural Network?
Layered mathematical nodes inspired by the brain — the foundation of modern AI.
COMPONENT 01
Nodes (Neurons)
Organized in layers: input → hidden → output. Each node computes a weighted sum of its inputs, then applies an activation function.
COMPONENT 02
Weighted Connections
Each connection Wij carries a weight. The network learns by adjusting these weights to minimize prediction error.
COMPONENT 03
Activation Function
Sigmoid σ(x) = 1/(1+e⁻ˣ) maps any value to (0,1). Introduces non-linearity — without it, stacked layers are just linear regression.
COMPONENT 04
Learning Algorithm
Backpropagation + Gradient Descent. Calculus (chain rule) tells us how much each weight contributed to the error so we can adjust it.
THEOREM
Universal Approximation
A neural network with even a single hidden layer (with enough neurons) can approximate any continuous function to arbitrary precision.
FOUNDATION OF
Modern Deep Learning
CNNs (images), RNNs (sequences), Transformers (ChatGPT) — all build on these same fundamentals. Master this and the rest unlocks.
Step 1 — Combination Function
netj = Σ Wij · xi
Weighted sum of all inputs including bias. x₀ = 1 always (acts like the intercept in regression).
Step 2 — Sigmoid Activation
σ(net) = 1 / (1 + e−net)
Output always in (0,1). Derivative: σ'(net) = σ(net)·(1−σ(net)) — essential for backprop.
Gradient Descent — Weight Update
wnew = wold − η · (∂SSE/∂w)
η (eta) = learning rate. Move weights downhill on the error surface to minimize SSE.
Backpropagation — Delta Rule
ΔWij = η · δj · xi
δj = error responsibility of node j. Propagated backward using the chain rule.

Backpropagation — Computing Weight Updates
Using the same worked example: ŷ=0.8327, y=1.0, η=0.1. Step through the delta calculations.
Output Node δ (delta)
δZ = oZ(1−oZ)(y−oZ)
Error responsibility for the output node. Combines the sigmoid derivative with the prediction error.
Hidden Node δ (delta)
δj = oj(1−oj) · Σ(Wjk·δk)
Error flows backward from downstream nodes k. No direct target — error is borrowed from the output.
Weight Update Rule
ΔWij = η · δj · xi
Adjust each weight proportionally to the learning rate × delta × input signal through that connection.
With Momentum
ΔW(t) = η·δ·x + α·ΔW(t−1)
α = momentum (typically 0.9). Adds inertia — accelerates convergence, smooths oscillations.

# CS655 – Neural Networks: Scikit-Learn MLPClassifier
# ═══════════════════════════════════════════════════════

from sklearn.neural_network import MLPClassifier
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report
from sklearn.datasets import load_breast_cancer
import numpy as np

# ── Load Dataset ─────────────────────────────────────
data = load_breast_cancer()
X, y = data.data, data.target

# ── Split (use training data ONLY for scaling!) ───────
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.20, random_state=42
)

# ── Feature Scaling (Min-Max or StandardScaler) ───────
scaler = StandardScaler()
X_train_s = scaler.fit_transform(X_train)  # fit on train only!
X_test_s  = scaler.transform(X_test)       # apply same scale to test

# ── Architecture 1: Shallow (1 hidden layer) ─────────
model_shallow = MLPClassifier(
    hidden_layer_sizes=(50,),      # 1 hidden layer, 50 neurons
    activation='relu',            # ReLU avoids vanishing gradient
    solver='adam',               # adaptive learning rate + momentum
    max_iter=300,
    random_state=42,
    validation_fraction=0.15,     # early stopping validation set
    early_stopping=True,          # stop when val loss stops improving
    n_iter_no_change=10           # patience = 10 epochs
)

# ── Architecture 2: Deeper (2 hidden layers) ─────────
model_deep = MLPClassifier(
    hidden_layer_sizes=(100, 50),  # 2 layers: 100 then 50 neurons
    activation='relu',
    solver='adam',
    learning_rate_init=0.001,     # Adam default starting LR
    max_iter=500,
    random_state=42,
    early_stopping=True,
    n_iter_no_change=15
)

# ── Train & Evaluate ──────────────────────────────────
for name, model in [("Shallow", model_shallow), ("Deep", model_deep)]:
    model.fit(X_train_s, y_train)
    y_pred = model.predict(X_test_s)
    acc = accuracy_score(y_test, y_pred)
    print(f"\n── {name} Network ──")
    print(f"Accuracy:  {acc:.4f}")
    print(f"Epochs:    {model.n_iter_}")
    print(classification_report(y_test, y_pred, target_names=data.target_names))

# CS655 – Neural Networks: TensorFlow / Keras
# ═══════════════════════════════════════════════════════

import tensorflow as tf
from tensorflow.keras import layers, callbacks
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split
from sklearn.datasets import load_breast_cancer
import matplotlib.pyplot as plt

# ── Data Prep ─────────────────────────────────────────
data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.20, random_state=42
)
scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)
X_test  = scaler.transform(X_test)

# ── Build Model ───────────────────────────────────────
model = tf.keras.Sequential([
    layers.Dense(100, activation='relu',
                 input_shape=(X_train.shape[1],)),  # input layer
    layers.Dropout(0.3),                           # 30% dropout
    layers.Dense(50,  activation='relu'),           # hidden layer 2
    layers.Dropout(0.2),
    layers.Dense(1,   activation='sigmoid')         # output: binary
])

model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)

model.summary()   # print architecture

# ── Callbacks: Early Stopping ─────────────────────────
early_stop = callbacks.EarlyStopping(
    monitor='val_loss',
    patience=10,            # wait 10 epochs before stopping
    restore_best_weights=True
)

# ── Train ─────────────────────────────────────────────
history = model.fit(
    X_train, y_train,
    epochs=200,
    validation_split=0.2,    # 20% of train → validation
    batch_size=32,
    callbacks=[early_stop],
    verbose=1
)

# ── Evaluate ──────────────────────────────────────────
test_loss, test_acc = model.evaluate(X_test, y_test, verbose=0)
print(f"\nTest Accuracy: {test_acc:.4f}")
print(f"Test Loss:     {test_loss:.4f}")

# ── Plot Training vs Validation Loss ──────────────────
plt.figure(figsize=(10, 4))
plt.subplot(1, 2, 1)
plt.plot(history.history['loss'],     label='Training Loss')
plt.plot(history.history['val_loss'], label='Validation Loss')
plt.title('Loss Curves'); plt.xlabel('Epoch'); plt.legend()

plt.subplot(1, 2, 2)
plt.plot(history.history['accuracy'],     label='Training Acc')
plt.plot(history.history['val_accuracy'], label='Validation Acc')
plt.title('Accuracy Curves'); plt.xlabel('Epoch'); plt.legend()
plt.tight_layout()
plt.show()

# CS655 – Sensitivity Analysis (Feature Importance)
# ═══════════════════════════════════════════════════════

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

def sensitivity_analysis(model, X_train, feature_names):
    """
    Measure relative influence of each input feature on model output.
    Procedure (from Slide 14):
      1. Create baseline: all features = mean value
      2. Vary each feature from min → max
      3. Record output range for each feature
      4. Larger range = higher influence
    """
    # Step 1: baseline observation (every feature at mean)
    x_mean = X_train.mean(axis=0).reshape(1, -1)
    baseline = model.predict_proba(x_mean)[0][1]

    results = []
    for i, feature in enumerate(feature_names):
        x_low  = x_mean.copy(); x_low[0, i]  = X_train[:, i].min()
        x_high = x_mean.copy(); x_high[0, i] = X_train[:, i].max()

        out_low  = model.predict_proba(x_low)[0][1]
        out_high = model.predict_proba(x_high)[0][1]

        influence = abs(out_high - out_low)
        results.append({
            'Feature': feature,
            'Low Output': round(out_low, 4),
            'High Output': round(out_high, 4),
            'Influence (Range)': round(influence, 4)
        })

    df = pd.DataFrame(results).sort_values('Influence (Range)', ascending=False)

    # Plot horizontal bar chart
    plt.figure(figsize=(10, 6))
    plt.barh(df['Feature'][::-1], df['Influence (Range)'][::-1], color='steelblue')
    plt.xlabel('Output Range (Influence)')
    plt.title('Sensitivity Analysis — Feature Importance')
    plt.tight_layout()
    plt.show()

    return df

# ── Run Sensitivity Analysis ───────────────────────────
# (assumes model_shallow and scaler from sklearn example above)
feature_names = load_breast_cancer().feature_names
df_importance = sensitivity_analysis(model_shallow, X_train_s, feature_names)
print(df_importance.to_string(index=False))

# ── 4. Output Denormalization (Regression) ────────────
#    x_pred = output × (x_max - x_min) + x_min
def denormalize(output, x_min, x_max):
    """Convert normalized network output back to original scale."""
    return output * (x_max - x_min) + x_min

Using This Demo in Class (~45 min)
1
Open Tab 01 — Overview (5 min)
Walk through the 6 concept cards. Point out the 4 formula cards at the bottom. Ask students: "Why can't a neural network just be a stack of linear equations?" (Answer: it collapses to a single linear model — activation functions are essential.)
2
Open Tab 02 — Feedforward (10 min)
Click Next → one step at a time. Have students calculate each step on paper BEFORE clicking — then verify. Watch the signal animate through the network. The values table updates in sync. Pause on Step 5 (Error = 0.1673) and ask: "How does the network know which weights caused this error?"
3
Open Tab 03 — Backpropagation (10 min)
Step through the 6 backprop steps. Emphasize Step 1 (δZ) — this is where error flows backward. Show how δA and δB get their error from downstream nodes, not directly from the target. The weight update table shows before/after values in green. Ask: "Why are the weight changes so tiny?"
4
Open Tab 04 — Training Demo (10 min)
First, show default settings — click ▶ Train and let it run. Then demonstrate: (a) Set η very high → watch loss bounce or diverge. (b) Set η very low → watch slow convergence. (c) Add dropout → compare regularized vs. unregularized. This makes the hyperparameter discussion concrete and memorable.
